# CHZZK Live Memo Marker 개인정보 처리방침

최종 업데이트: 2026-04-12

이 확장 프로그램은 CHZZK(치지직) 시청 중 사용자가 입력/가져온 메모 데이터를 브라우저 로컬 저장소에 보관하기 위해 동작합니다.

## 수집 및 저장하는 정보

- 메모/카테고리 기록
- 세션 정보(스트리머명, 제목, 타임라인)
- 단축키 설정
- VOD 연결 정보
- 플레이어에 표시할 버튼(메모·연결) 표시 여부
- 라이브 카테고리 / 방송 제목(방제) 자동 감지 사용 여부

## 저장 위치

- 모든 데이터는 사용자의 브라우저 내부 `chrome.storage.local`에 저장됩니다.

## 외부 전송

- 본 확장 프로그램은 사용자 메모/기록 내용을 별도 서버로 전송하지 않습니다.
- 네트워크 접근은 CHZZK 웹페이지 범위(`https://chzzk.naver.com/*`)로 제한됩니다. 라이브 시각·카테고리는 **페이지 DOM 및 페이지에 포함된 스크립트**만 사용하며, 치지직 Open API 서버·`api.chzzk.naver.com` 등으로 **별도 `fetch`를 보내지 않습니다.**

## 권한 사용 목적

- `tabs`: 팝업을 열었을 때 **현재 활성 탭의 URL**만 읽어, 해당 탭이 치지직 VOD인지 판별하고 타임라인 연결에 사용합니다. URL을 외부로 전송하지 않습니다.
- `windows`: 사용자가 선택할 때만 **같은 확장 UI(`popup.html`)를 별도 브라우저 창**으로 엽니다. 다른 사이트 창을 읽거나 조작하지 않습니다.
- `storage`: 메모/세션/설정 저장
- `downloads`: 사용자가 요청한 TXT 내보내기 파일 저장

## 제3자 제공

- 사용자 데이터를 제3자에게 판매/공유하지 않습니다.

## 사용자 제어 방법

- 확장 프로그램 삭제 시 브라우저에 저장된 해당 확장 데이터는 함께 제거될 수 있습니다.
- 필요 시 확장 팝업에서 세션/기록을 직접 삭제할 수 있습니다.
