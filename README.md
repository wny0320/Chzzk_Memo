# CHZZK Live Memo Marker

CHZZK 라이브/VOD 시청 중 메모와 카테고리를 기록하고, VOD 타임라인 마커로 다시 이동할 수 있게 도와주는 Chrome Extension입니다.

> 이 프로젝트는 **오픈소스**이며, **CHZZK(치지직) 공식 제품이 아닌 비공식 확장 프로그램**입니다.
>
> 이 저장소는 **바이브 코딩(Vibe Coding)** 방식으로 빠르게 실험·개선하며 개발한 프로젝트입니다.
>
> 개발 과정에서 **Cursor AI를 적극 활용한 AI-assisted 바이브 코딩 프로젝트**임을 명시합니다.

## 주요 기능

- 라이브/다시보기 시점에 맞춰 메모 저장
- 라이브 **카테고리 / 방송 제목(방제)** 자동 기록 및 변경 감지(**일반 화면만** — 아래 권장·제한 참고). **확장 팝업에서 자동 감지 ON/OFF** 를 끌 수 있습니다. VOD 타임라인 마커에는 **「카테고리」/「카테고리 변경」** 과 **카테고리 이름**이 함께 표시됩니다.
- VOD 진행바 타임라인 마커 표시
- 마커 클러스터 목록에서 원하는 시점으로 즉시 이동
- 댓글의 타임라인(`HH:MM:SS`, `H:MM:SS`, `MM:SS`, `M:SS`)을 세션으로 가져오기
- 팝업에서 스트리머/세션 단위 기록 조회, 수정, 삭제
- TXT보내기/가져오기 및 클립보드 복사
- 메모 단축키 커스터마이징

## 카테고리·방제 자동 기록 — 권장 사용

- **처음 시청할 때** 치지직 **일반(기본) 레이아웃**에서, 화면에 **방송 경과 시각·카테고리 이름** 등이 로딩되어 보이기 시작한 뒤에 **전체 화면** 또는 **넓은 화면 보기(시어터)** 로 바꾸는 것을 권장합니다. 시어터·전체 화면으로 먼저 들어가면 카테고리가 DOM에 잘 안 붙어 **첫 자동 기록이 빠지거나 늦어질 수 있습니다.**
- **방송 중 카테고리·방제가 바뀐 뒤**에도 자동 감지는 **일반 레이아웃에서만** 안정적으로 동작합니다. 변경이 있었다면 **잠시 일반 모드로 돌아온 시점**에 확장이 페이지에서 값을 읽고, 그때 **「카테고리 변경」** 등으로 세션에 남깁니다. 넓은 화면·전체 화면만 쓰면 바뀐 내용이 **기록되지 않을 수 있습니다.**

## 제한 사항

- **라이브 카테고리 / 방송 제목 자동 감지**는 위와 같이 **기본(일반) 레이아웃**을 전제로 합니다. **전체 화면** 또는 **넓은 화면 보기(시어터)** 에서는 사이트 쪽에 카테고리 문구·링크가 DOM에 나타나지 않는 경우가 많아, 이 확장의 자동 감지는 **의도적으로 기대하지 마세요.** 방송 경과 시각 등 일부 정보는 **같은 탭 안의 페이지(DOM·화면에 노출된 텍스트)** 만 읽어 보강합니다. **치지직 [공개 Open API](https://chzzk.gitbook.io/chzzk/chzzk-api/live)의 `/open/v1/...` 호출이나, 웹이 내부적으로 쓰는 `api.chzzk.naver.com/service/...` 등으로의 별도 `fetch`는 하지 않습니다**([아래](#api-및-외부-네트워크) 참고).

## API 및 외부 네트워크

- 이 확장은 **`https://chzzk.naver.com` 탭 외부로 카테고리·메타데이터를 가져가기 위한 `fetch`를 보내지 않습니다.** (공개 문서의 Open API는 앱 등록·Client 인증이 필요한 `/open/v1/...` 경로이며, 시청 중 페이지에서 쓰는 `.../service/v2/channels/.../live-detail` 등은 문서에 포함된 공개 규약이 아닙니다.)
- 스트림 메타데이터·경과 시각·카테고리 등은 **`https://chzzk.naver.com` 페이지를 사용자가 연 상태에서**, 브라우저가 이미 받아 둔 화면(DOM)과 페이지 내 스크립트/텍스트에서만 해석합니다.
- **별도 백엔드·분석 서버로 사용자 메모나 시청 데이터를 보내지 않습니다.** (저장은 `chrome.storage.local`만 사용합니다.)

## 권한

- `storage`: 메모/세션/설정 로컬 저장
- `downloads`: TXT보내기 저장
- `tabs`: 팝업에서 현재 활성 탭 URL로 VOD 여부 확인(타임라인 연결)
- `windows`: 팝업 UI를 **별도 창**으로 열 때(툴바 팝업은 바깥 클릭 시 닫힘)
- `host_permissions`: `https://chzzk.naver.com/*`

## 개인정보 및 데이터 처리

- 본 확장은 사용자 기록 데이터를 `chrome.storage.local`에 저장합니다.
- 별도의 외부 서버로 전송하지 않으며, **API 키·토큰·원격 설정을 받아오는 통신도 하지 않습니다.**

## 설치(개발자 모드)

1. Chrome에서 `chrome://extensions` 열기
2. 우측 상단 **개발자 모드** 활성화
3. **압축해제된 확장 프로그램을 로드합니다** 클릭
4. 이 저장소를 클론했다면 **`Chzzk_Memo` 폴더(저장소 루트)** 를 선택합니다.
